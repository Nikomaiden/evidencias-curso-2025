# 📁 Portafolio de Evidencias

## Descripción

Este repositorio tiene como propósito almacenar y organizar las evidencias 
generadas a lo largo del curso, incluyendo análisis, documentos, actividades 
y demás productos académicos desarrollados durante el proceso de aprendizaje.

Cada evidencia se encuentra organizada por carpetas según la semana o actividad 
correspondiente, permitiendo un seguimiento claro y trazable del progreso 
académico.

## Estructura del repositorio
```
📦 portafolio-evidencias/
├── 📂 semana-1/
├── 📂 semana-2/
├── 📂 semana-3/
└── 📄 README.md
```
